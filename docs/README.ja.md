# Prism 💍
> ターミナルで完結するGitHub PRレビューTUI

## 特徴
- ブラウザを開かずにPRレビューが完結
- Vimライクなキーバインドで高速操作
- 差分表示、コメント投稿、マージまで対応
- CIステータスの確認
- レビュー進捗の自動保存

## インストール
```bash
  go install github.com/urabexon/prism@latest
```
## 使い方
1. カレントリポジトリで起動
```bash
  prism
```
2. リポジトリを指定
```bash
  prism owner/repo
```
3. キーバインド
#### PR一覧
| キー      | 操作     |
| ------- | ------ |
| `j/k`   | 移動     |
| `Enter` | PRを開く  |
| `m`     | マージ    |
| `c`     | CIチェック |
| `q`     | 終了     |
#### 差分表示
| キー      | 操作       |
| ------- | -------- |
| `j/k`   | スクロール    |
| `n/N`   | 次/前のhunk |
| `V`     | 範囲選択     |
| `space` | レビュー済み   |
| `esc`   | 戻る       |

## 動作環境
- Go 1.21+
- GitHub CLI (gh) 認証
